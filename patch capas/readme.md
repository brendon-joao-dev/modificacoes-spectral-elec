# O que eu sei até o momento:

Para ter capa no seus jogos você tem que fazer o seguinte:
- 1º - Dentro da pasta do emulador onde ficam as roms (Ex: roms/gba) você cria uma pasta chamada "images"
- 2º - A sua rom e a capa dela devem ter o exato mesmo nome incluindo letras maiúsculas e minúsculas
- 3º - Dentro da sua pasta roms você deve criar um arquivo chamado "gamelist.xml"
- 4º - Nesse arquivo você criará uma estrutura semelhante a do arquivo gamelist.xml aqui contido
- 5º - Para cada jogo você cria um jogo novo e coloca as informações dele, sendo as essenciais para a capa:
    - Em "path" o caminho para o jogo (Ex: ./MegaMan Zero I (BR).zip)
    - Em "image" o caminho para a capa (Ex: ./images/MegaMan Zero I (BR).png)

*OBS: Se existirem mais capas do que roms nenhuma delas funcionará!!! Porém pode haver mais roms do que capas*
